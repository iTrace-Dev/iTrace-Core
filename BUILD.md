### SUPPORTED ARCHITECTURES
	* x64
	
### REQUIREMENTS
* Git
* Visual Studio 2017 (C#)
	* Windows Application
	* Nuget
	* .Net Framework 4.6.1
* Tobii Pro .Net SDK v1.5.0.942 x64 (provided by Nuget)

### WINDOWS BUILD
* Clone iTrace-Core repo
* Open VS Solution
* Build Solution (This should also force nuget to acquire the Tobii SDK package)
* Run Application
